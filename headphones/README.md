
## Sections Included

1. **Header**: Sticky navigation with mobile menu toggle
2. **Hero Section**: Main banner with call-to-action
3. **Features Section**: Key product benefits
4. **Products Section**: Product showcase with pricing
5. **Testimonials**: Customer reviews
6. **Footer**: Contact information and links

## Responsive Behavior

- **Mobile (< 480px)**: Stacked layout, hamburger menu, larger touch targets
- **Tablet (481px - 768px)**: Two-column grids, adjusted spacing
- **Desktop (> 769px)**: Full layout with optimized reading width

## How to Use

1. Clone the repository
2. Open `index.html` in any modern web browser
3. Resize the browser window to see responsive behavior

## Browser Compatibility

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Font Awesome Icons
- Google Fonts

## Color Scheme

- Primary: `#FF5555` (hover/active states)
- Secondary: `#333333` (text, buttons)
